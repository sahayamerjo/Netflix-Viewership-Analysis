# Netflix-Viewership-Analysis
This project analyzes Netflix global content released in 2023 to understand how factors such as content type, language, release timing, seasonality, and holidays influence viewership hours. The analysis uses Python, Pandas, and Matplotlib to uncover trends and visualize audience engagement.
